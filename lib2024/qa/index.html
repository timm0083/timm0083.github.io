  1. Qus: What are the three componets of Web?

    Ans: HTML, CSS, and Javascript.

  2. Qus: Define version control and give one example.

    Ans: Version control is a system that records all the changes made to a file(s) over time so that you can see all the different versions of the file(s) when needed. GitHub, for example, is a version control software.

  3. Qus: What tag is used to create the title of a web page? Give an example.

    Ans: <title> is the tag used to create a title of a web page. For example, <title>QA Assignment</title>

  4. Qus: What tag is used to create a paragraph in html? Give an example.

    Ans: <p> is the tag used to create a paragraph in html. For example, <p>This is a paragraph</p>

  5. Qus: Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason?

    Ans: <p> is the odd tag out because it is a block-level element and the tags <span>, <a>, and <i> are inline elements.

  6. Qus: What are the tags required to create a navigation list in HTML?

    Ans: To create a navigation list in HTML you need to use the tags <nav> <ul> <li><a href="#">Home</a></li> </ul> </nav>

  7. Qus: What element is used to create images? Name its attributes.

    Ans: The <img> tag is used to create images. The attributes for the <img> tag are: alt, crossorigin, height, ismap, loading, longdesc, referrerpolicy, sizes, src, srcset, usemap, width.

  8. Qus: Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason

    Ans: <b> is the odd one out in these tags because the rest of these tags are used to make a table in HTML, but <b> is not a tag used to make a table. The <b> tag is used to make text bold.

  9. Qus: Give an example of color picker input.

    Ans: <label for="favcolor">Select your favorite color:</label> <input type="color" id="favcolor" name="favcolor" value="#ff0000">

  10. Qus: What is the use of name and value attributes? Give two examples

    Ans: The name attribute is used to specify the name of an <input> element. The value attribute is the property that defines the input data.

  11. Qus: Give an example of inline style.

    Ans: <h1 style="color:blue;text-align:center;">This is a heading</h1> <p style="color:red;">This is a paragraph.</p>

  12. Qus: How would you link an external style sheet? Give example.

    Ans: You can link an external style sheet in and HTML document by using the <link> tag inside of the <head> <head> <link rel="stylesheet" type="text/css" href="css/style.css"> </head>
